# Laporan RT

<div className="role-badge ketua-rt">
  ðŸ‘” Role: Ketua RT
</div>

Panduan melihat dan mengekspor laporan RT berisi ringkasan demografi, mobilitas, statistik rumah, dan aktivitas RT.

## Akses

<div className="step-container">
  <div className="step-number">1</div>
  <div className="step-content">
    <h3>Buka Menu Layanan</h3>
    <p>Dari dashboard, pilih tab <strong>Layanan</strong> lalu pilih <strong>Laporan RT</strong>.</p>
  </div>
</div>

<div className="step-container">
  <div className="step-number">2</div>
  <div className="step-content">
    <h3>Tampilan Ringkasan</h3>
    <p>Halaman menampilkan filter wilayah dan pilihan periode (Bulanan, Triwulan, Tahunan) serta kartu ringkasan di bagian atas.</p>
    <ul>
      <li><strong>Total Penduduk</strong></li>
      <li><strong>Total KK</strong></li>
      <li><strong>Total Kelahiran</strong></li>
      <li><strong>Total Kematian</strong></li>
    </ul>
    <p>Gunakan filter wilayah untuk melihat laporan per jalan atau gang.</p>
  </div>
</div>

<div className="step-container">
  <div className="step-number">3</div>
  <div className="step-content">
    <h3>Periode Laporan</h3>
    <p>Pilih periode laporan: <strong>Bulanan</strong>, <strong>Triwulan</strong>, atau <strong>Tahunan</strong>. Pilihan periode akan mempengaruhi semua grafik dan tabel.</p>
  </div>
</div>

<div className="step-container">
  <div className="step-number">4</div>
  <div className="step-content">
    <h3>Analisis yang Ditampilkan</h3>
    <p>Halaman berisi beberapa kelompok analisis dengan grafik interaktif dan angka ringkasan.</p>
  </div>
</div>

<div className="step-container">
  <div className="step-number">4.1</div>
  <div className="step-content">
    <h3>Statistik Demografi</h3>
    <ul>
      <li><strong>Distribusi Usia</strong>: diagram donat yang membagi penduduk menjadi Anak, Remaja, Dewasa, Lansia; menyertakan angka dan persentase.</li>
      <li><strong>Jenis Kelamin</strong>: diagram donat (L / P) dengan rincian angka.</li>
      <li><strong>Status Perkawinan</strong>: grafik batang dengan kategori Belum Menikah, Menikah, Cerai Hidup, Cerai Mati dan angka masing-masing.</li>
    </ul>
  </div>
</div>

<div className="step-container">
  <div className="step-number">4.2</div>
  <div className="step-content">
    <h3>Mobilitas Vital</h3>
    <ul>
      <li><strong>Mutasi Keluarga</strong>: bar chart (Masuk / Keluar) per periode.</li>
      <li><strong>Kelahiran & Kematian</strong>: bar chart menunjukkan jumlah kelahiran dan kematian pada periode terpilih.</li>
    </ul>
  </div>
</div>

<div className="step-container">
  <div className="step-number">4.3</div>
  <div className="step-content">
    <h3>Statistik Rumah</h3>
    <ul>
      <li><strong>Hunian</strong>: diagram donat (Berpenghuni / Kosong) dengan rincian angka.</li>
      <li><strong>Kepemilikan</strong>: diagram donat (Kontrak / Tetap) dengan rincian angka.</li>
    </ul>
  </div>
</div>

<div className="step-container">
  <div className="step-number">4.4</div>
  <div className="step-content">
    <h3>Kegiatan & Laporan RT</h3>
    <p>Bar chart yang membandingkan jumlah Surat, Laporan, dan Kegiatan sesuai periode, lengkap dengan angka rinci.</p>
  </div>
</div>


<div className="step-container">
  <div className="step-number">5</div>
  <div className="step-content">
    <h3>Export / Unduh</h3>
    <p>Klik tombol <strong>Export</strong> untuk mengunduh laporan lengkap. Pilih format: <strong>PDF</strong> untuk laporan cetak atau <strong>Excel/CSV</strong> untuk data mentah, lalu klik <strong>Unduh</strong>.</p>
  </div>
</div>

:::tip Best Practice
- Gunakan filter wilayah untuk membuat laporan granular per gang atau jalan.
- Pilih periode yang sesuai untuk analisis tren (mis. triwulan untuk pemantauan menengah).
- Pastikan data sumber diperbarui secara berkala (pendaftaran warga, mutasi, kelahiran, kematian).
:::

:::warning Perhatian
- Pastikan data sumber terkini sebelum mengekspor atau memutuskan berdasarkan laporan.
:::
