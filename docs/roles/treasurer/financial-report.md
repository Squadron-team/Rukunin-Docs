 # Riwayat Transaksi & Unduh Laporan

 <div className="role-badge treasurer">
   ðŸ’° Role: Bendahara
 </div>

 Halaman Riwayat Transaksi berisi semua catatan pemasukan dan pengeluaran dalam periode yang dipilih. Gunakan halaman ini untuk meninjau transaksi, memfilter berdasarkan tipe/periode, dan mengekspor laporan.

 <div class="step-container">
   <div class="step-number">1</div>
   <div class="step-content">
     <h3>Akses Menu</h3>
     <p>Dari dashboard, pilih tab <strong>Transaksi</strong> lalu klik <strong>Riwayat Transaksi</strong>.</p>
   </div>
 </div>

 <div class="step-container">
   <div class="step-number">2</div>
   <div class="step-content">
     <h3>Tampilan Daftar</h3>
     <p>Daftar menampilkan baris transaksi dengan kolom utama: <strong>Jenis Transaksi</strong>, <strong>Catatan</strong>, <strong>Jumlah</strong>, dan <strong>Waktu</strong>.</p>
     <p>Ikon panah menunjukkan arah transaksi: panah ke bawah (warna hijau) = pemasukan; panah ke atas (warna merah) = pengeluaran.</p>
   </div>
 </div>

 <div class="step-container">
   <div class="step-number">3</div>
   <div class="step-content">
     <h3>Filter</h3>
     <p>Gunakan filter untuk menyaring data:</p>
     <ul>
       <li>
            <strong>Tipe</strong>. Semua Transaksi / Pemasukan / Pengeluaran
        </li>
       <li>
        <strong>Periode</strong>. Bulanan / Triwulan / Tahunan (pilih rentang atau preset)
       </li>
     </ul>
   </div>
 </div>

 <div class="step-container">
   <div class="step-number">4</div>
   <div class="step-content">
     <h3>Unduh Laporan</h3>
     <p>
        Klik <strong>Unduh Laporan Transaksi</strong> untuk mengekspor hasil filter saat ini. Pilihan format ekspor:
        <ul>
            <li>PDF</li>
            <li>Excel (XLSX)</li>
            <li>CSV</li>
        </ul>
     </p>
   </div>
 </div>

 <div class="step-container">
   <div class="step-number">5</div>
   <div class="step-content">
     <h3>Melihat Detail Transaksi</h3>
     <p>Untuk melihat detail, klik salah satu baris transaksi. Halaman detail menampilkan:</p>
     <ul>
       <li>Jenis Transaksi</li>
       <li>Jumlah</li>
       <li>Keterangan / Catatan</li>
       <li>Jenis Pembayaran (Manual / Transfer)</li>
       <li>Tanggal & Waktu</li>
       <li>Lampiran bukti pembayaran (gambar yang bisa di-zoom)</li>
     </ul>
   </div>
 </div>

 :::tip Praktik Baik
 - Gunakan filter periode untuk fokus pada rentang yang relevan sebelum mengunduh.
 - Simpan versi PDF untuk dokumentasi resmi dan XLSX/CSV untuk analisis lebih lanjut.
 :::
