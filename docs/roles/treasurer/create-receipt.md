# Membuat Template Kwitansi

<div className="role-badge treasurer">
  ðŸ’° Role: Bendahara
</div>

Halaman ini digunakan untuk membuat **template kwitansi** yang akan dipakai saat Bendahara membuat kwitansi iuran warga. Sistem akan otomatis mengisi data keluarga dan nomor kwitansi saat template dipakai.

<div class="step-container">
  <div class="step-number">1</div>
  <div class="step-content">
    <h3>Akses Menu</h3>
    <p>Dari dashboard, pilih tab <strong>Iuran</strong> lalu klik <strong>Buat Kwitansi</strong>.</p>
  </div>
</div>

<div class="step-container">
  <div class="step-number">2</div>
  <div class="step-content">
    <h3>Isi Detail Template</h3>
    <p>Di halaman template, isi atau unggah elemen berikut:</p>
    <ul>
      <li><strong>Logo</strong> â€” format JPG/PNG, maksimal 5 MB</li>
      <li><strong>Tanda tangan Bendahara</strong> â€” format JPG/PNG, maksimal 5 MB</li>
      <li><strong>Judul kwitansi</strong> (mis. "Kwitansi Iuran RT 01")</li>
      <li><strong>Nama penerima</strong> (opsional, dapat diisi otomatis saat dipakai)</li>
      <li><strong>Catatan kaki</strong> (mis. informasi rekening atau kontak)</li>
    </ul>
    <p>Catatan: saat template digunakan, sistem akan mengisi otomatis data keluarga dan nomor kwitansi.</p>
  </div>
</div>

<div class="step-container">
  <div class="step-number">3</div>
  <div class="step-content">
    <h3>Preview</h3>
    <p>Gunakan tombol <strong>Preview</strong> untuk melihat tampilan kwitansi dengan data contoh sebelum menyimpan template. Periksa logo, tanda tangan, dan tata letak.</p>
  </div>
</div>

<div class="step-container">
  <div class="step-number">4</div>
  <div class="step-content">
    <h3>Simpan Template</h3>
    <p>Jika sudah sesuai, klik <strong>Simpan Template</strong>. Template akan tersedia di daftar template dan bisa dipilih saat membuat kwitansi untuk warga.</p>
  </div>
</div>

:::tip Praktik Baik
- Gunakan logo resolusi sedang agar file tetap ringan.
:::

:::warning Perhatian
- Pastikan file logo dan tanda tangan berformat JPG/PNG dan tidak lebih dari 5 MB.
- Perubahan pada template tidak mengubah kwitansi yang sudah diterbitkan sebelumnya.
:::
