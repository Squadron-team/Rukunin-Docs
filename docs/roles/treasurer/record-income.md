# Mencatat Pemasukan

<div className="role-badge treasurer">
  ğŸ’° Role: Bendahara
</div>

Halaman untuk mencatat semua pemasukan kas, termasuk iuran, donasi, dan pendapatan lain.
<div class="step-container">
  <div class="step-number">1</div>
  <div class="step-content">
    <h3>Akses Menu</h3>
    <p>Dari dashboard, pilih tab <strong>Transaksi</strong> lalu klik <strong>Pemasukan</strong>.</p>
  </div>
</div>

<div class="step-container">
  <div class="step-number">2</div>
  <div class="step-content">
    <h3>Isi Form Pemasukan</h3>
    <p>Lengkapi data pemasukan sebagai berikut:</p>
    <ul>
      <li>
        <strong>Sumber Pemasukan</strong>, pilih salah satu: Bantuan/Hibah, Donasi Kegiatan, Pendapatan Lain
      </li>
      <li>
        <strong>Nominal</strong>, masukkan jumlah (angka)
      </li>
      <li>
      <strong>Tanggal</strong>, pilih tanggal transaksi
      </li>
      <li>
      <strong>Bukti Transaksi</strong>, pilih salah satu:
        <ul>
          <li>Foto struk (ambil foto dengan kamera atau pilih dari galeri)</li>
          <li>Tunai (centang jika transaksi tunai tanpa struk)</li>
        </ul>
      </li>
      <li>
        <strong>Keterangan / Catatan</strong> (opsional)
      </li>
    </ul>
  </div>
</div>

<div class="step-container">
  <div class="step-number">3</div>
  <div class="step-content">
    <h3>Simpan</h3>
    <p>Setelah lengkap, klik <strong>Simpan</strong>. Transaksi akan tercatat dan muncul di ringkasan serta laporan keuangan.</p>
  </div>
</div>

:::tip Praktik
- Lampirkan foto bukti jika ada untuk mempermudah audit.
- Untuk transaksi tunai, beri catatan jelas (penerima, tujuan) di kolom keterangan.
:::
