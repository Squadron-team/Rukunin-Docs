## Detail Informasi RT

Klik salah satu RT untuk melihat detail lengkap:

### Informasi Umum
- **Nomor RT** dan wilayah cakupan
- **Ketua RT** dan kontak
- **Pengurus RT** (Sekretaris, Bendahara)
- **Batas wilayah** geografis

### Statistik RT
- Total warga aktif
- Jumlah Kartu Keluarga
- Rasio pembayaran iuran
- Jumlah kegiatan bulan ini
- Jumlah laporan pending

### Keuangan RT
- Saldo kas RT
- Total iuran terkumpul bulan ini
- Pengeluaran bulan ini
- Tunggakan iuran

## Monitor Kinerja RT

Gunakan fitur **Kinerja RT** untuk evaluasi:

### Metrik Kinerja

| Indikator | Target | Keterangan |
|-----------|--------|------------|
| **Pembayaran Iuran** | â‰¥ 85% | Persentase warga yang bayar tepat waktu |
| **Kehadiran Rapat** | â‰¥ 75% | Partisipasi warga dalam rapat |
| **Waktu Respons** | â‰¤ 24 jam | Respons terhadap laporan warga |
| **Update Data** | Mingguan | Frekuensi update data warga |
| **Kegiatan** | â‰¥ 2/bulan | Jumlah kegiatan yang dilaksanakan |

### Filter & Periode

Anda bisa filter kinerja berdasarkan:
- Periode waktu (bulanan, triwulan, tahunan)
- Kategori tertentu (keuangan, kegiatan, layanan)
- Perbandingan antar RT

### Laporan Kinerja

Generate laporan kinerja dalam format:
- **PDF** - Untuk dokumentasi
- **Excel** - Untuk analisis lebih lanjut
- **Dashboard Visual** - Untuk presentasi

## Tips Koordinasi RT

:::tip Best Practice
- **Rapat koordinasi rutin** - Minimal 1x sebulan dengan semua Ketua RT
- **Grup komunikasi** - Buat grup WhatsApp atau Telegram untuk koordinasi cepat
- **Sharing best practice** - Bagikan cara-cara efektif dari RT yang berkinerja baik
- **Bantuan mutual** - Fasilitasi RT untuk saling membantu
- **Apresiasi** - Berikan penghargaan kepada RT yang berkinerja baik
:::

## Troubleshooting

### RT tidak aktif melaporkan kegiatan

**Solusi**:
1. Hubungi Ketua RT untuk menanyakan kendala
2. Berikan pelatihan penggunaan aplikasi jika perlu
3. Tunjuk pendamping jika Ketua RT kesulitan

### Data RT tidak terupdate

**Solusi**:
1. Reminder kepada Ketua RT
2. Cek apakah ada masalah teknis
3. Bantu input data jika RT kewalahan

### Konflik antar warga di satu RT

**Solusi**:
1. Fasilitasi mediasi
2. Libatkan tokoh masyarakat jika perlu
3. Dokumentasikan penyelesaian untuk pembelajaran
