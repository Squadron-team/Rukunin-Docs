## Tips Pengelolaan Akun

:::tip Best Practice

- **Gunakan autentikasi biometrik** untuk keamanan maksimal
- **Perbarui data profil** secara berkala
- **Pilih bahasa sesuai preferensi**
- **Logout jika menggunakan perangkat bersama**
- **Gunakan password yang kuat dan unik**

:::

## Troubleshooting

### Tidak bisa login

**Solusi**:

1. Periksa koneksi internet
2. Gunakan fitur "Lupa Password"
3. Hubungi admin jika akun diblokir

### Autentikasi biometrik gagal

**Solusi**:

1. Pastikan sensor bersih dan jari sesuai
2. Gunakan opsi "USE PASSWORD"
3. Restart aplikasi
