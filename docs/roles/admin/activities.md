## Tampilan Kalender

- Menampilkan bulan aktif (misalnya: Desember 2025)
- Tanggal dengan titik oranye menandakan ada kegiatan terjadwal
- Tanggal yang dipilih akan menampilkan detail kegiatan di bawah kalender

## Filter Kegiatan

Gunakan filter untuk menyaring kegiatan berdasarkan wilayah:

| Filter | Deskripsi |
|--------|-----------|
| **Semua** | Tampilkan semua kegiatan |
| **RW** | Hanya kegiatan tingkat RW |
| **RT** | Hanya kegiatan tingkat RT |

### Panel Filter Tambahan

- **RW**: Pilih RW tertentu atau "Semua RW"
- **RT**: Pilih RT tertentu atau "Semua RT"
- Klik <strong>Terapkan Filter</strong> untuk menyaring hasil

## Tips Pemantauan Kegiatan

:::tip Best Practice

- **Gunakan filter RW/RT** untuk fokus pada wilayah tertentu
- **Pantau partisipasi warga** melalui jumlah peserta
- **Catat kegiatan rutin** untuk evaluasi tahunan
- **Koordinasi dengan pengurus** jika ada kegiatan yang perlu dukungan

:::

## Troubleshooting

### Kegiatan tidak muncul di kalender

**Solusi**:

1. Pastikan tanggal sudah dipilih
2. Periksa filter RW/RT
3. Refresh halaman

### Jumlah peserta tidak sesuai

**Solusi**:

1. Buka daftar peserta
2. Pastikan data sudah sinkron
3. Hubungi support jika masih bermasalah
