# Mulai Cepat

Selamat datang di dokumentasi **Rukunin** - aplikasi manajemen RT/RW yang memudahkan administrasi lingkungan Anda.

## Apa itu Rukunin?

Rukunin adalah aplikasi mobile yang dirancang khusus untuk membantu pengelolaan Rukun Tetangga (RT) dan Rukun Warga (RW) di Indonesia. Aplikasi ini menyediakan berbagai fitur untuk memudahkan administrasi, komunikasi, dan pengelolaan keuangan lingkungan.

## Role di Aplikasi

Aplikasi Rukunin memiliki 6 role berbeda dengan akses dan fitur yang disesuaikan:

| Role | Deskripsi | Akses |
|------|-----------|-------|
| **Admin** | Mengelola sistem secara keseluruhan | Penuh |
| **Ketua RW** | Mengelola data dan aktivitas tingkat RW | Tingkat RW |
| **Ketua RT** | Mengelola data dan aktivitas tingkat RT | Tingkat RT |
| **Sekretaris** | Mengelola surat dan dokumen | Administrasi |
| **Bendahara** | Mengelola keuangan dan iuran | Keuangan |
| **Warga** | Menggunakan layanan RT/RW | Terbatas |

## Panduan Berdasarkan Role

Pilih panduan sesuai dengan role Anda:

- [**Admin**](/docs/roles/admin/overview) - Panduan lengkap untuk Administrator
- [**Ketua RW**](/docs/roles/ketua-rw/overview) - Panduan untuk Ketua RW
- [**Ketua RT**](/docs/roles/ketua-rt/overview) - Panduan untuk Ketua RT
- [**Sekretaris**](/docs/roles/secretary/overview) - Panduan untuk Sekretaris
- [**Bendahara**](/docs/roles/treasurer/overview) - Panduan untuk Bendahara
- [**Warga**](/docs/roles/resident/overview) - Panduan untuk Warga
